# User Input Screens

This document outlines all available user input screens in the application, alongside a short description of its purpose.

This is used to create all test related screenings.

## User Login Screen

The user login screen allows two options:

```bash
Welcome to InCollege!
Log In
Create a new Account
Enter your choice:
```

## Log In Account Screen
The create account screen prompts the user to enter their details to create a new account:

```bash
Please enter your username:
Please enter your password:
You have successfully logged in.
Welcome, [Username]!
<insert main main menu here>
```

## Main Menu Screen
```bash
1. Search for a job
2. Find someone you know
3. Learn a new Skill
4. Logout
Enter your choice:
```

Where 1 and 2 are unimplemented, 3 brings you to the Skill Learning Screen Selection, and 4 ends the program.
